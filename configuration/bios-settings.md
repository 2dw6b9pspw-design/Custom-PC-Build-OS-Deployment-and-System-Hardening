# BIOS Configuration

This document outlines the BIOS configuration and validation steps performed after system assembly to ensure system stability, hardware compatibility, and readiness for operating system installation.

---

## 1. BIOS Access and Hardware Verification

- Accessed BIOS during initial boot using the designated function key
- Verified detection of installed hardware:
  - CPU: AMD Ryzen 5 7600X3D
  - Memory: 32GB DDR5
  - Storage: Samsung 990 Pro NVMe SSD

---

## 2. BIOS Version Management

- Verified BIOS version upon first system boot
- Updated BIOS to the latest stable release provided by the motherboard manufacturer
- Confirmed successful BIOS update and clean system reboot

---

## 3. Memory Configuration

- Enabled manufacturer-recommended memory profile (EXPO/XMP)
- Verified applied memory speed and timings
- Confirmed system stability following configuration changes

---

## 4. Boot Configuration

- Verified NVMe SSD was recognized as a bootable device
- Set system boot mode to UEFI
- Confirmed boot order in preparation for operating system installation

---

## 5. Thermal and Fan Configuration

- Verified CPU cooler and case fans were detected
- Reviewed default fan curves to ensure adequate cooling behavior
- Confirmed idle CPU temperatures remained within expected operating ranges

---

## 6. Integrated Graphics Configuration

- Verified integrated graphics were enabled for display output
- Confirmed HDMI/DisplayPort functionality during POST and BIOS navigation

---

## 7. Security and Stability Checks

- Confirmed default firmware security settings were enabled
- Verified system voltages and power delivery values were within expected ranges
- Reviewed POST status indicators for errors or warnings

---

## 8. Final Validation

- Saved BIOS configuration changes
- Performed multiple system reboots to confirm stability
- Prepared system for operating system installation
